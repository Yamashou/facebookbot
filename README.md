facebookbot
====

Grow up and make more smart your own bot.

## Requirement
- Golang environment
- Cabocha

## Install

    $ go get github.com/m2mtu/facebookbot

### Deploy to Heroku

    $ heroku buildpacks:add https://github.com/masawada/heroku-buildpack-linuxbrew
    $ git push heroku master
